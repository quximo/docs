## CLI Tool

Quximo comes with a **command-line interface (CLI)** to manage services and infrastructure directly from your local environment. All CLI commands update configuration files in your project’s `.quximo/` folder, which Quximo uses to orchestrate deployments and platform settings.

Read more at https://docs.quximo.com/cli/quick-start for a detailed overview.
